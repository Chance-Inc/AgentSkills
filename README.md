# AgentSkills

Internal AgentSkills for Chance AI.

## Skills

| Skill | Description |
|-------|-------------|
| [gh-issue-writer](./gh-issue-writer) | Write high-quality GitHub issues from vague symptoms, with full Chance AI 3-layer stack context |

## Usage

Drop a `.skill` file into `~/.openclaw/skills/` to install, or clone this repo and symlink the skill folder.
